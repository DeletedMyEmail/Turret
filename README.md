## Code Details

The project was initializied via the VsCode extention PlatformIO. It's an easy way of working with Arduino stuff in VsCode.

## Requierements

- ESP32 (e.g. NodeMCU-32s)
- 3 servo motors
- 3D Printer
- Mosfet (e.g. IRFZ44N)
